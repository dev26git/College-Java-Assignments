Write a Java application that will be able to add, subtract, multiply, divide, compare, convert to floating point,
and find absolute value for rational numbers, with the exception handling.
Identify and create different exceptions that will be important in your project.






A rational no. is in the form of a/b where a, b are integers and b != 0.
Rational no.s can be added subtracted multiplied divided compared and converted to double and floating points.
Absolute values of rational numbers can also be found.

Accept 2 nos. from KLBD through CLA & output results of operations. Use Exception Handling.
